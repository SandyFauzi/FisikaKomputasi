## Tugas 1 — Rekonstruksi Hambatan Udara

### Penurunan Model Matematika

Pada gerak peluru **tanpa hambatan udara**, satu-satunya gaya yang bekerja adalah gravitasi. Ketika hambatan udara dimasukkan dengan model gaya kuadratik:

$$\vec{F}_d = -k v^2 \hat{v}$$

di mana $k$ adalah koefisien hambat udara (kg/m), $v$ adalah **kecepatan total**, dan $\hat{v}$ adalah vektor satuan arah gerak.

#### Langkah 1 — Uraikan kecepatan total

Karena gerak berlangsung dalam 2D, kecepatan total adalah:

$$v = |\vec{v}| = \sqrt{v_x^2 + v_y^2}$$

#### Langkah 2 — Uraikan gaya hambat ke komponen x dan y

Vektor satuan arah gerak:

$$\hat{v} = \frac{\vec{v}}{|\vec{v}|} = \left(\frac{v_x}{v},\ \frac{v_y}{v}\right)$$

Sehingga komponen gaya hambat:

$$F_{d,x} = -kv^2 \cdot \frac{v_x}{v} = -k\,v\,v_x$$

$$F_{d,y} = -kv^2 \cdot \frac{v_y}{v} = -k\,v\,v_y$$

#### Langkah 3 — Terapkan Hukum Newton II pada tiap sumbu

**Sumbu x** (tidak ada gaya lain selain drag):

$$m\,\frac{dv_x}{dt} = -k\,v\,v_x$$

$$\boxed{\frac{dv_x}{dt} = -\frac{k}{m}\,v\,v_x}$$

**Sumbu y** (gravitasi + drag):

$$m\,\frac{dv_y}{dt} = -mg - k\,v\,v_y$$

$$\boxed{\frac{dv_y}{dt} = -g - \frac{k}{m}\,v\,v_y}$$

#### Langkah 4 — Diskritisasi ke Metode Euler

Dari skema Euler $f^{n+1} = f^n + \Delta t \cdot \dot{f}^n$, diperoleh sistem update:

| Variabel | Update Numerik |
|----------|----------------|
| $x$ | $x^{n+1} = x^n + v_x^n \,\Delta t$ |
| $y$ | $y^{n+1} = y^n + v_y^n \,\Delta t$ |
| $v_x$ | $v_x^{n+1} = v_x^n - \dfrac{k}{m}\,v^n\,v_x^n\,\Delta t$ |
| $v_y$ | $v_y^{n+1} = v_y^n + \left(-g - \dfrac{k}{m}\,v^n\,v_y^n\right)\Delta t$ |

#### Pengaruh terhadap Lintasan

Tanpa hambatan udara, lintasan membentuk **parabola simetris**. Dengan hambatan udara:
- Jangkauan horizontal berkurang drastis karena $v_x$ terus direm oleh drag
- Sisi **turun lebih curam** dari sisi naik, parabola menjadi asimetris
- Ketinggian puncak juga berkurang karena $v_y$ awal juga terhambat

### Kode Python — Tugas 1

```python
import numpy as np
import matplotlib.pyplot as plt

# ─── Parameter ───────────────────────────────────────────
g     = 9.81    # m/s²
m     = 1.0     # kg
v0    = 30.0    # m/s  — kecepatan awal
theta = 45.0    # derajat — sudut elevasi
k     = 0.05    # kg/m  — koefisien hambat udara
dt    = 0.005   # s     — time step Euler

def euler_peluru(k_coef):
    """
    Simulasi gerak peluru dengan metode Euler.
    k_coef = 0   → tanpa hambatan udara
    k_coef > 0   → dengan hambatan udara Fd = -k*v²
    """
    rad = np.radians(theta)

    # ── Kondisi Awal (t = 0) ─────────────────────────────
    x  = 0.0
    y  = 0.0
    vx = v0 * np.cos(rad)
    vy = v0 * np.sin(rad)

    xs, ys = [x], [y]

    while True:
        v = np.sqrt(vx**2 + vy**2)      # kecepatan total

        # ── Percepatan (Newton II + air drag) ───────────
        ax = -(k_coef / m) * v * vx     # sumbu x: hanya drag
        ay = -g - (k_coef / m) * v * vy # sumbu y: gravitasi + drag

        # ── Update Euler ─────────────────────────────────
        x  = x  + vx * dt
        y  = y  + vy * dt
        vx = vx + ax * dt
        vy = vy + ay * dt

        xs.append(x)
        ys.append(y)

        # ── Syarat Batas Dirichlet: y = 0 (tanah) ───────
        if y < 0:
            break

    return np.array(xs), np.array(ys)

# ─── Jalankan simulasi ───────────────────────────────────
x_nd, y_nd = euler_peluru(k_coef=0.0)    # tanpa drag
x_d,  y_d  = euler_peluru(k_coef=k)     # dengan drag

# ─── Visualisasi ─────────────────────────────────────────
fig, ax = plt.subplots(figsize=(10, 5))

ax.plot(x_nd, y_nd, 'b-', linewidth=2,
        label=f'Tanpa drag (k=0)  →  Range = {x_nd[-1]:.1f} m')
ax.plot(x_d,  y_d,  'r-', linewidth=2,
        label=f'Dengan drag (k={k})  →  Range = {x_d[-1]:.1f} m')

ax.axhline(0, color='brown', linewidth=2)
ax.fill_between([0, x_nd[-1]+5], -2, 0, color='sienna', alpha=0.3)
ax.set_xlim(0, x_nd[-1]+5)
ax.set_ylim(-2, y_nd.max()+5)
ax.set_xlabel('Jarak Horizontal (m)')
ax.set_ylabel('Ketinggian (m)')
ax.set_title(f'Tugas 1: Lintasan Peluru Dengan dan Tanpa Hambatan Udara (θ={theta}°)')
ax.legend()
ax.grid(True, alpha=0.4)

plt.tight_layout()
plt.savefig('tugas1_hambatan_udara.png', dpi=150)
plt.show()
```

## Tugas 2 — Analisis Jarak Maksimum

### Penjelasan Teoritis

**Tanpa hambatan udara**, jangkauan horizontal (range) diperoleh secara analitik:

$$R = \frac{v_0^2 \sin 2\theta}{g}$$

$R$ maksimum saat $\sin 2\theta = 1$, yaitu $2\theta = 90°$, sehingga:

$$\theta_{opt} = 45°$$

**Dengan hambatan udara**, tidak ada solusi analitik tertutup. Namun secara fisis, hambatan udara **menguntungkan sudut lebih rendah** karena:

1. Sudut rendah → waktu terbang lebih singkat → drag bekerja lebih sedikit
2. Sudut tinggi → waktu terbang panjang → drag menguras lebih banyak energi kinetik horizontal
3. Efek drag pada $v_x$ terakumulasi sepanjang waktu terbang — semakin lama di udara, semakin besar total perlambatan

Akibatnya, $\theta_{opt}$ bergeser ke **bawah 45°** dan semakin rendah seiring $k$ semakin besar.

### Kode Python — Tugas 2

```python
import numpy as np
import matplotlib.pyplot as plt

g   = 9.81
m   = 1.0
v0  = 30.0
dt  = 0.005
k   = 0.05

def range_peluru(theta_deg, k_coef):
    """Hitung jangkauan horizontal gerak peluru."""
    rad = np.radians(theta_deg)
    x, y = 0.0, 0.0
    vx = v0 * np.cos(rad)
    vy = v0 * np.sin(rad)

    while True:
        v  = np.sqrt(vx**2 + vy**2)
        ax = -(k_coef / m) * v * vx
        ay = -g - (k_coef / m) * v * vy

        x  += vx * dt
        y  += vy * dt
        vx += ax * dt
        vy += ay * dt

        if y < 0:
            break
    return x

# ─── Sweep sudut 1° sampai 89° ───────────────────────────
thetas         = np.arange(1, 89, 1)
range_nodrag   = [range_peluru(th, 0.0) for th in thetas]
range_drag     = [range_peluru(th, k)   for th in thetas]

# ─── Cari sudut optimal ──────────────────────────────────
theta_opt_nd = thetas[np.argmax(range_nodrag)]
theta_opt_d  = thetas[np.argmax(range_drag)]

print(f"Sudut optimal TANPA drag : {theta_opt_nd}°  (teoritis: 45°)")
print(f"Sudut optimal DENGAN drag: {theta_opt_d}°")
print(f"Pergeseran sudut optimal : {theta_opt_nd - theta_opt_d}° ke bawah")

# ─── Visualisasi ─────────────────────────────────────────
fig, ax = plt.subplots(figsize=(10, 5))

ax.plot(thetas, range_nodrag, 'b-', linewidth=2, label='Tanpa drag (k=0)')
ax.plot(thetas, range_drag,   'r-', linewidth=2, label=f'Dengan drag (k={k})')

ax.axvline(theta_opt_nd, color='blue', linestyle='--', linewidth=1.2,
           label=f'θ optimal k=0: {theta_opt_nd}°')
ax.axvline(theta_opt_d,  color='red',  linestyle='--', linewidth=1.2,
           label=f'θ optimal k={k}: {theta_opt_d}°')

ax.set_xlabel('Sudut Elevasi θ (derajat)')
ax.set_ylabel('Jangkauan / Range (m)')
ax.set_title('Tugas 2: Jangkauan Maksimum vs Sudut Elevasi')
ax.legend()
ax.grid(True, alpha=0.4)

plt.tight_layout()
plt.savefig('tugas2_range_vs_sudut.png', dpi=150)
plt.show()
```

## Tugas 3 — Gerak Jatuh Bebas & Terminal Velocity

### Penurunan Terminal Velocity

Pada gerak jatuh bebas dengan hambatan udara kuadratik, persamaan gerak (sumbu y ke bawah positif):

$$m\frac{dv}{dt} = mg - kv^2$$

$$\frac{dv}{dt} = g - \frac{k}{m}v^2$$

**Terminal velocity** tercapai saat percepatan $= 0$, yaitu gaya drag seimbang dengan gravitasi:

$$0 = g - \frac{k}{m}v_{term}^2$$

$$\frac{k}{m}v_{term}^2 = g$$

$$v_{term}^2 = \frac{mg}{k}$$

$$\boxed{v_{term} = \sqrt{\frac{mg}{k}}}$$

#### Interpretasi Fisis

- Saat $v < v_{term}$: drag lebih kecil dari gravitasi → benda **masih dipercepat**
- Saat $v = v_{term}$: drag = gravitasi → percepatan = 0 → **kecepatan konstan**
- Benda tidak pernah melebihi $v_{term}$ secara mandiri

Ini analog dengan rangkaian RC: kapasitor charging mendekati $V_{supply}$ secara asimtotik — benda mendekati $v_{term}$ namun tidak pernah melampaui.

#### Diskritisasi Euler

$$v^{n+1} = v^n + \left(g - \frac{k}{m}(v^n)^2\right)\Delta t$$

$$y^{n+1} = y^n + v^n \,\Delta t$$

### Kode Python — Tugas 3

```python
import numpy as np
import matplotlib.pyplot as plt

g   = 9.81
m   = 1.0
k   = 0.1     # kg/m — koefisien hambat
dt  = 0.005   # s

# ─── Terminal velocity analitik ──────────────────────────
v_term = np.sqrt(m * g / k)
print(f"Terminal velocity analitik: sqrt({m}*{g}/{k}) = {v_term:.4f} m/s")

# ─── Simulasi Euler: jatuh bebas dengan hambatan udara ───
v = 0.0    # kecepatan awal
y = 0.0    # posisi awal (diukur ke bawah)
t = 0.0

v_list, a_list, t_list = [v], [g], [t]

while y < 300:   # jatuh sampai 300 m
    a  = g - (k / m) * v**2    # percepatan saat ini
    v  = v + a * dt             # update kecepatan (Euler)
    y  = y + v * dt             # update posisi
    t  = t + dt

    v_list.append(v)
    a_list.append(a)
    t_list.append(t)

v_arr = np.array(v_list)
a_arr = np.array(a_list)
t_arr = np.array(t_list)

# ─── Cari kapan 99% v_term tercapai ─────────────────────
idx99  = np.argmax(v_arr >= 0.99 * v_term)
t_99   = t_arr[idx99] if idx99 > 0 else None
print(f"99% v_term = {0.99*v_term:.4f} m/s tercapai pada t ≈ {t_99:.2f} s")
print(f"Percepatan saat itu: {a_arr[idx99]:.4f} m/s²  (mendekati 0)")

# ─── Visualisasi ─────────────────────────────────────────
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 8), sharex=True)

# Plot v(t)
ax1.plot(t_arr, v_arr, 'g-', linewidth=2, label='v(t) numerik')
ax1.axhline(v_term, color='gold', linestyle='--', linewidth=1.5,
            label=f'v_term = {v_term:.2f} m/s')
if t_99:
    ax1.axvline(t_99, color='red', linestyle=':', linewidth=1.2,
                label=f'99% v_term @ t={t_99:.1f} s')
    ax1.plot(t_99, 0.99*v_term, 'ro', markersize=8)
ax1.set_ylabel('Kecepatan v (m/s)')
ax1.set_title(f'Tugas 3: Gerak Jatuh Bebas dengan Hambatan Udara (k={k}, m={m} kg)')
ax1.legend()
ax1.grid(True, alpha=0.4)

# Plot a(t) — percepatan mendekati nol saat terminal velocity
ax2.plot(t_arr, a_arr, 'r-', linewidth=2, label='a(t) = dv/dt')
ax2.axhline(0, color='gray', linestyle='--', linewidth=1)
ax2.set_xlabel('Waktu t (s)')
ax2.set_ylabel('Percepatan a (m/s²)')
ax2.set_title('Percepatan → 0 saat Terminal Velocity tercapai')
ax2.legend()
ax2.grid(True, alpha=0.4)

plt.tight_layout()
plt.savefig('tugas3_terminal_velocity.png', dpi=150)
plt.show()
```

## Tugas 4 — Visualisasi Komparatif 3 Nilai k

### Penjelasan

Dengan memvariasikan $k$, kita dapat melihat secara langsung bagaimana besar koefisien hambat udara mempengaruhi jangkauan dan bentuk lintasan. Tiga kasus yang dibandingkan:

| Nilai $k$ | Interpretasi Fisis | Analogi |
|-----------|-------------------|---------|
| $k = 0.01$ | Hambatan sangat kecil (benda padat, berat) | Bola besi |
| $k = 0.05$ | Hambatan sedang | Bola tenis |
| $k = 0.15$ | Hambatan besar (benda ringan, luas penampang besar) | Bola pingpong |

Semakin besar $k$:
- Jangkauan semakin pendek
- Lintasan semakin tidak simetris (sisi turun lebih curam)
- Puncak ketinggian lebih rendah dan lebih dekat ke titik awal

### Kode Python — Tugas 4

```python
import numpy as np
import matplotlib.pyplot as plt

g     = 9.81
m     = 1.0
v0    = 30.0
theta = 45.0
dt    = 0.005

def euler_peluru(k_coef):
    """Simulasi gerak peluru. Return arrays x dan y."""
    rad = np.radians(theta)
    x, y = 0.0, 0.0
    vx = v0 * np.cos(rad)
    vy = v0 * np.sin(rad)
    xs, ys = [x], [y]

    while True:
        v  = np.sqrt(vx**2 + vy**2)
        ax = -(k_coef / m) * v * vx
        ay = -g - (k_coef / m) * v * vy

        x  += vx * dt
        y  += vy * dt
        vx += ax * dt
        vy += ay * dt

        xs.append(x)
        ys.append(y)

        if y < 0:
            break

    return np.array(xs), np.array(ys)

# ─── Konfigurasi 3 variasi k ─────────────────────────────
configs = [
    {'k': 0.00,  'label': 'k = 0.00  (tanpa drag)',      'color': 'gray',   'ls': '--'},
    {'k': 0.01,  'label': 'k = 0.01  (hambatan kecil)',  'color': '#4fc3f7','ls': '-'},
    {'k': 0.05,  'label': 'k = 0.05  (hambatan sedang)', 'color': '#69f0ae','ls': '-'},
    {'k': 0.15,  'label': 'k = 0.15  (hambatan besar)',  'color': '#ff7043','ls': '-'},
]

fig, ax = plt.subplots(figsize=(11, 6))

x_max = 0
y_max = 0

for cfg in configs:
    xs, ys = euler_peluru(cfg['k'])
    rng    = xs[-1]
    label  = f"{cfg['label']}  →  Range = {rng:.1f} m"
    ax.plot(xs, ys, color=cfg['color'], linestyle=cfg['ls'],
            linewidth=2.2, label=label)
    print(f"  {label}")
    x_max = max(x_max, rng)
    y_max = max(y_max, ys.max())

ax.axhline(0, color='brown', linewidth=2)
ax.fill_between([0, x_max+5], -2, 0, color='sienna', alpha=0.3)
ax.set_xlim(0, x_max+5)
ax.set_ylim(-2, y_max+5)
ax.set_xlabel('Jarak Horizontal (m)')
ax.set_ylabel('Ketinggian (m)')
ax.set_title(f'Tugas 4: Komparatif Lintasan Peluru — 3 Variasi Koefisien Hambat Udara k\n'
             f'(θ = {theta}°, v₀ = {v0} m/s, m = {m} kg)')
ax.legend(loc='upper right')
ax.grid(True, alpha=0.4)

plt.tight_layout()
plt.savefig('tugas4_komparatif_k.png', dpi=150)
plt.show()
```
