## Tugas 1 — Rekonstruksi Hambatan Udara

### Penurunan Model Matematika

Pada gerak peluru **tanpa hambatan udara**, satu-satunya gaya yang bekerja adalah gravitasi. Ketika hambatan udara dimasukkan dengan model gaya kuadratik:

$$\vec{F}_d = -k v^2 \hat{v}$$

di mana $k$ adalah koefisien hambat udara (kg/m), $v$ adalah **kecepatan total**, dan $\hat{v}$ adalah vektor satuan arah gerak.

#### Langkah 1 — Uraikan kecepatan total

$$v = |\vec{v}| = \sqrt{v_x^2 + v_y^2}$$

#### Langkah 2 — Uraikan gaya hambat ke komponen x dan y

$$F_{d,x} = -kv^2 \cdot \frac{v_x}{v} = -k\,v\,v_x \qquad F_{d,y} = -kv^2 \cdot \frac{v_y}{v} = -k\,v\,v_y$$

#### Langkah 3 — Terapkan Hukum Newton II tiap sumbu

$$\boxed{\frac{dv_x}{dt} = -\frac{k}{m}\,v\,v_x} \qquad \boxed{\frac{dv_y}{dt} = -g - \frac{k}{m}\,v\,v_y}$$

#### Langkah 4 — Diskritisasi Euler

| Variabel | Update Numerik |
|----------|----------------|
| $x$ | $x^{n+1} = x^n + v_x^n \,\Delta t$ |
| $y$ | $y^{n+1} = y^n + v_y^n \,\Delta t$ |
| $v_x$ | $v_x^{n+1} = v_x^n - \dfrac{k}{m}v^n v_x^n\,\Delta t$ |
| $v_y$ | $v_y^{n+1} = v_y^n + \left(-g - \dfrac{k}{m}v^n v_y^n\right)\Delta t$ |

### Kode Python — Tugas 1 (Animasi)

```python
import numpy as np
import matplotlib.pyplot as plt
import matplotlib.patches as mpatches
from matplotlib.animation import FuncAnimation

# ─── Parameter ───────────────────────────────────────────────────
g     = 9.81    # m/s²
m     = 1.0     # kg
v0    = 30.0    # m/s
theta = 45.0    # derajat
k     = 0.05    # koefisien hambat udara (kg/m)
dt    = 0.005   # time step Euler

# ─── Fungsi simulasi Euler ────────────────────────────────────────
def euler_peluru(k_coef):
    rad = np.radians(theta)
    x, y = 0.0, 0.0
    vx = v0 * np.cos(rad)
    vy = v0 * np.sin(rad)
    xs, ys = [x], [y]
    while True:
        v   = np.sqrt(vx**2 + vy**2)
        ax  = -(k_coef / m) * v * vx
        ay  = -g - (k_coef / m) * v * vy
        x  += vx * dt;  y  += vy * dt
        vx += ax * dt;  vy += ay * dt
        xs.append(x);   ys.append(y)
        if y < 0:
            break
    return np.array(xs), np.array(ys)

x_nd, y_nd = euler_peluru(0.0)   # tanpa drag
x_d,  y_d  = euler_peluru(k)     # dengan drag

# Ambil setiap N frame saja
N    = 4
xnd_s, ynd_s = x_nd[::N], y_nd[::N]
xd_s,  yd_s  = x_d[::N],  y_d[::N]
n_frames = max(len(xnd_s), len(xd_s))

# ─── Setup Figure ────────────────────────────────────────────────
fig, ax = plt.subplots(figsize=(11, 5))
fig.patch.set_facecolor('#0f1117')
ax.set_facecolor('#1a1d27')

xmax = x_nd[-1] * 1.08

ax.set_ylabel('Ketinggian (m)', color='#aaa')
ax.set_title(f'Tugas 1 — Lintasan Peluru: Dengan vs Tanpa Hambatan Udara  (θ={theta}°, v₀={v0} m/s)',
             color='white', pad=10)
ax.tick_params(colors='#888')
for sp in ax.spines.values(): sp.set_edgecolor('#333')
ax.grid(True, color='#2a2d3a', linestyle='--', lw=0.5, alpha=0.7)

# Ground
ax.axhline(0, color='#6d4c28', lw=2)
ax.fill_between([0, xmax], -2, 0, color='#3e2a10', alpha=0.6)

# ── Static full trajectory (faint, drawn once) ───────────────────
ax.plot(x_nd, y_nd, color='#4fc3f7', lw=1, alpha=0.18, zorder=1)
ax.plot(x_d,  y_d,  color='#ff7043', lw=1, alpha=0.18, zorder=1)

# ── Animated trails ──────────────────────────────────────────────
trail_nd, = ax.plot([], [], color='#4fc3f7', lw=1.8, alpha=0.55,
                    label=f'Tanpa drag (k=0)  →  R={x_nd[-1]:.1f} m', zorder=2)
trail_d,  = ax.plot([], [], color='#ff7043', lw=1.8, alpha=0.55,

dot_d,  = ax.plot([], [], 'o', color='#ff7043', ms=8, zorder=5)

# ── Info text ────────────────────────────────────────────────────

    return trail_nd, trail_d, dot_nd, dot_d, info_txt
```

## Tugas 2 — Analisis Jarak Maksimum

### Penjelasan Teoritis

**Tanpa hambatan udara**: $R = \dfrac{v_0^2 \sin 2\theta}{g}$ → maksimum di $\theta = 45°$

**Dengan hambatan udara**: tidak ada solusi analitik. Sudut optimal $< 45°$ karena:
- Sudut tinggi → waktu terbang lebih lama → drag menguras lebih banyak energi horizontal
- Sudut rendah → waktu terbang lebih singkat → total kehilangan energi lebih kecil

Animasi di bawah menembakkan peluru dari sudut 5° hingga 85° secara berurutan, langsung terlihat sudut mana yang menghasilkan jangkauan terpanjang.

## Tugas 3 — Gerak Jatuh Bebas & Terminal Velocity

### Penurunan Terminal Velocity

$$m\frac{dv}{dt} = mg - kv^2 \implies \frac{dv}{dt} = g - \frac{k}{m}v^2$$

Saat $a = 0$:

$$0 = g - \frac{k}{m}v_{term}^2 \implies \boxed{v_{term} = \sqrt{\frac{mg}{k}}}$$

**Diskritisasi Euler:** $v^{n+1} = v^n + \left(g - \dfrac{k}{m}(v^n)^2\right)\Delta t$

### Kode Python — Tugas 3 (Animasi)

```python
import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation
from matplotlib.gridspec import GridSpec

g    = 9.81
m    = 1.0
k    = 0.1
dt   = 0.05

v_term = np.sqrt(m * g / k)
print(f"Terminal velocity analitik: {v_term:.4f} m/s")

# ─── Simulasi jatuh bebas ─────────────────────────────────────────
v, y, t = 0.0, 0.0, 0.0
vs, ys, ts, accs = [v], [y], [t], [g]

    y += v * dt
    t += dt

ts    = np.array(ts)

             color='white', fontsize=11)
```

## Tugas 4 — Visualisasi Komparatif 3 Nilai k

### Penjelasan

| $k$ | Hambatan | Analogi fisis |
|-----|----------|---------------|
| 0.01 | Sangat kecil | Bola logam |
| 0.05 | Sedang | Bola tenis |
| 0.15 | Besar | Bola pingpong |

Semakin besar $k$: jangkauan lebih pendek, lintasan makin asimetris.

```python
                 'range': xs[-1], **cfg})

n_frames = max(len(t['xs']) for t in traj)

lines, dots = [], []
for t in traj:
    ln, = ax.plot([], [], color=t['color'], lw=2.2,
                  label=f"{t['label']}  →  R={t['range']:.1f} m")
    dt_, = ax.plot([], [], 'o', color=t['color'], ms=9, zorder=5)

          fontsize=9, loc='upper right')

                   color='#ccc', fontsize=9, va='top', fontfamily='monospace')

    for ln in lines: ln.set_data([], [])
    for dt_ in dots: dt_.set_data([], [])
    time_txt.set_text('')

def update(frame):
    artists = []
    for i, t in enumerate(traj):
        idx = min(frame, len(t['xs']) - 1)
        xs_f = t['xs'][:idx+1]
        ys_f = np.maximum(t['ys'][:idx+1], 0)

        dots[i].set_data([t['xs'][idx]], [max(t['ys'][idx], 0)])

    t_sim = frame * N * dt
    time_txt.set_text(f't = {t_sim:.2f} s')
    artists.append(time_txt)
    return artists
```
